Step 1: Open Slack App Directory
Go to the official GitHub app for Slack. (https://slack.github.com/?utm_source=chatgpt.com)
Click Add to Slack.
Select your Slack workspace.
Click Allow.

Step 2: Subscribe a Slack Channel to a Repository
Create a Slack channel where you want notifications and type:
/github subscribe owner/repository
Example:
/github subscribe microsoft/vscode

Step 3: Connect Your GitHub Account
Sign in to GitHub.
Authorize the Slack app to access your GitHub account or organization.


Once installed, Slack will automatically send notifications for:
Commits
Pull Requests
Issues
Releases
GitHub Actions workflows
